# Food-101 미니 프로젝트

23102006 김하진

## 1. 프로젝트 개요

본 프로젝트는 Food-101 데이터셋을 사용하여 음식 이미지를 분류하는 합성곱 신경망(CNN) 모델을 구현하고,  
기본 CNN 모델과 전이 학습을 적용한 ResNet18 모델의 성능을 비교하는 것을 목표로 한다.

Food-101 데이터셋은 101가지 음식 종류로 구성되어 있으며,  
음식의 종류가 다양하고 외형이 비슷한 경우가 많아 이미지 분류 연습용 데이터셋으로 많이 사용된다.

---

## 2. 데이터셋 설명

- 데이터셋: Food-101
- 클래스 수: 101개
- 전체 이미지 수: 101,000장
- 각 클래스당 약 1,000장 이미지

모든 이미지는 160×160 크기로 변환하였으며,  
ImageNet에서 사용되는 평균과 표준편차를 이용해 정규화를 수행하였다.

---

## 3. 모델 설계 및 구현

### 3.1 기본 CNN 모델 (Baseline)

기본 모델로 간단한 CNN 구조를 사용하였다.  
세 개의 합성곱 층과 맥스 풀링 층을 사용하였으며,  
마지막에는 두 개의 완전 연결층을 통해 분류를 수행하였다.

- 학습 방식: 처음부터 학습
- 옵티마이저: Adam
- 학습률: 0.001
- 배치 크기: 16
- 학습 에폭 수: 2

학습 결과는 다음과 같다.

- 학습 정확도: 약 1.0%
- 검증 정확도: 약 0.9%

Food-101은 101개의 클래스를 가지므로,  
기본 CNN 모델은 거의 랜덤 예측에 가까운 성능을 보였다.

---

### 3.2 ResNet18 모델 (전이 학습)

모델 성능을 개선하기 위해 ImageNet으로 사전 학습된 ResNet18 모델을 사용하였다.  
마지막 분류층을 Food-101에 맞게 101개 출력으로 수정하였다.

- 전이 학습 적용
- 옵티마이저: Adam
- 학습률: 0.0001
- 배치 크기: 16
- 학습 에폭 수: 3

학습 결과는 다음과 같다.

- 학습 정확도: 약 63.6%
- 검증 정확도: 약 60.7%

기본 CNN 모델과 비교했을 때 성능이 크게 향상되었다.

---

## 4. 평가 및 분석

### 4.1 학습 곡선 분석

기본 CNN 모델은 학습 정확도와 검증 정확도가 거의 증가하지 않았다.  
반면 ResNet18 모델은 학습이 진행됨에 따라 정확도가 꾸준히 증가하였다.

이를 통해 전이 학습이 Food-101과 같은 복잡한 데이터셋에서 효과적임을 확인할 수 있었다.

---

### 4.2 혼동 행렬 분석

혼동 행렬을 통해 모델이 어떤 음식들을 헷갈려 하는지 확인하였다.

- 면 요리나 튀김 음식처럼 외형이 비슷한 음식들에서 오분류가 자주 발생하였다.
- 색감이나 담는 방식이 비슷한 밥 요리도 서로 혼동되는 경우가 있었다.
- 피자나 햄버거처럼 모양이 뚜렷한 음식은 비교적 정확하게 분류되었다.

---

### 4.3 실패 사례 분석

잘못 분류된 이미지들을 분석한 결과 다음과 같은 원인을 확인할 수 있었다.

- 조명으로 인한 색상 변화
- 복잡한 배경으로 인한 음식 형태 인식 오류
- 같은 음식이라도 형태가 매우 다양한 경우

이러한 요인들이 모델의 성능에 영향을 미치는 것을 확인하였다.

---

## 5. 결론

본 프로젝트에서는 기본 CNN 모델과 ResNet18 모델을 비교하였다.

기본 CNN 모델은 Food-101 데이터셋을 학습하기에 구조적으로 한계가 있었으며,  
전이 학습을 적용한 ResNet18 모델은 짧은 학습 시간에도 높은 성능을 보였다.

이를 통해 이미지 분류 문제에서  
모델 구조와 사전 학습의 중요성을 확인할 수 있었다.

---
